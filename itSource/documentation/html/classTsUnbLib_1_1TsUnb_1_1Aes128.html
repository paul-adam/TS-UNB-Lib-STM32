Detailed Description

Implementation of AES-128 for ETSI TS 103 357 TS-UNB.

This class implemets the AES-128 encryption for ETSI TS 103 357 according to https://nvlpubs.nist.gov/nistpubs/FIPS/NIST.FIPS.197.pdf

This class uses special memory optimization for AVR microprocessors, e.g. to support the ATmega328p.
