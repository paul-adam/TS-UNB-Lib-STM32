Detailed Description

template<uint16_t SYMBOL_RATE_MULT = 48>
class TsUnbLib::STM32::STM32TsUnb< SYMBOL_RATE_MULT >

Platform dependent TS-UNB implementation for STM32 based systems.

This class implements all platform dependent methods for TS-UNB. It mainly offer SPI communication and a timer to generate the TS-UNB symbol clock.

Parameters
SYMBOL_RATE_MULTTS-UNB, symbol rate in multiples of 49.591064453125, set for 48 for 2380.371sym/s and 8 for 396.729sym/s. For higher rates the clock divider of the timer may have to be adjusted.
