Comparison between two float values.

This macro function may be called to compare two float values. The two values are said to be equal, if the difference is less than 0.0000001.

Parameters
[in]<strong>LEFT</strong>The first value to compare
[in]<strong>RIGHT</strong>The second value to compare
Returns
true, if the difference of both values is less than 0.0000001
false else
